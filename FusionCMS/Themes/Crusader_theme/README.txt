? HOW TO EDIT AND MAKE YOUR OWN LOGO
-----------------------------------------------------------------

- Open logo.psd via Photoshop CS5 or newer version and edit all of the colored text layers, make sure 
  you dont resize any of the text layers. Layers for the server name are colored in orange.
  Once you are ready with the text layers, press "SHIFT + CTRL + ALT + S" and save the image as .png, 
  transperency must be checked!
  Replace your logo with the one in the theme folder (Fusion_CMS\application\themes\crusader_theme\images\).



? SLIDER IMAGES
-----------------------------------------------------------------

- Open the folder "SLIDER IMAGES" and you will find 5 psd files :

   - slider_image_1.psd
   - slider_image_2.psd
   - slider_image_3.psd
   - slider_image_4.psd
   - slider_image_5.psd

  You can Open and edit every slider for your needs.
  Edit the text layers marked with green color and move the text if you want.
  Do not change the size of the images, just save them as .jpg (SHIFT + CTRL + ALT + S).



? REGISTER & VOTE BANNERS
-----------------------------------------------------------------
- You can find these .psd files in the fodler "PSD FILES" :

  - register_banner.psd
  - vote_banner.psd

  Here you must type in your text 2 times , one for normal state of the banner and then for the hover, 
  text layers for normal state are marked with red and for hover with green, make sure you dont move the text layers.
  
  Once you are done, just save the image as .png (SHIFT + CTRL + ALT + S), transperency must be checked!
  Replace the edited images with the original one in the images folder of the theme 
  (Fusion_CMS\application\themes\crusader_theme\images\).



? USER PANEL NEW BUTTONS
-----------------------------------------------------------------
- User panel buttons are already saved, you just have to replace the new one with older.
  Open "UCP Buttons" folder, copy all of the images inside, and replace them in --> Fusion_CMS\application\modules\ucp\images !
  If you want other buttons you can use the included .psd file (ucp_button.psd placed in PSD FILES folder).
