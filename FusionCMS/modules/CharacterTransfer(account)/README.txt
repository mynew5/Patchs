
#### INSTALLATION ####

 1. Extract the contents of the archive usually on the desktop.

 2. Copy the contents of the folder Content into the root folder of your copy of FusionCMS.
 
 3. Open the folder SQL from the extracted files from the module archive and import the sql file into your FusionCMS database.

 4. Go to the Admin Panel -> Menu Links.
    4. Click the "Create link" button.
    4. Step 1. Enter title for the link (Character Transfer etc...)
    4. Step 2. Enter URL like so: character_transfer
    4. step 3. Select one of the two possible menus Top or Side and submit.

 5. Navigate to "User groups & permissions" on the Admin Panel side menu.

 6. Give your groups permissions to the Character Transfer Module ACP Page.
    6. Click "Groups".
    6. Select your group usually "Owner" and click the pencil button (Edit).
    6. Locate the section "Roles (?)".
    6. Locate "Character Transfer (Account)" in that secontion and thick the two permissions (view and administrate).

 6. You are done.
